This is a database display and searching application, which takes input from a tsv file that can be edited/modified as much as desired.

It simply lists the information provided to it, and can do many functional tasks with that information.

This includes:

- Sorting alphabetically
- Sorting by year
- Searching by Name
- Searching by Year


In this specific case, we have a decent-sized database of artist information and music facts, all explorable.

Enjoy!
